Selenium for Automation Testing

Job Title: Automation Tester

As an Automation Tester using Selenium, you'll develop and execute automated test scripts to ensure the quality of software applications. You'll work closely with development teams to identify and fix issues before software release.

Responsibilities

  • Develop and maintain automated test scripts using Selenium.
  • Execute test cases and analyze test results.
  • Collaborate with developers to identify and resolve defects.
  • Enhance and update test scripts to adapt to application changes.
  • Participate in test planning and provide feedback on test coverage.
  • Generate test reports and document test cases and results.

Scope of Opportunity

This role offers the opportunity to work on automated testing frameworks and improve software quality. It provides a pathway to advanced roles in quality assurance, test automation architecture, or software development engineering in test (SDET).
