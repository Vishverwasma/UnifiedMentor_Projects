Linux & Git Workflow

Job Title: DevOps Engineer / Systems Administrator

As a DevOps Engineer or Systems Administrator, you'll work with Linux-based systems and manage source code using Git. You'll be responsible for ensuring smooth development workflows and maintaining system environments.

Responsibilities

  • Manage and maintain Linux servers and systems.
  • Implement and manage Git workflows for version control.
  • Automate deployment processes and manage CI/CD pipelines.
  • Monitor system performance and troubleshoot issues.
  • Collaborate with development teams to streamline development and deployment.

Scope of Opportunity

This role offers the chance to work on system administration and automation projects. It provides opportunities to advance into senior DevOps or infrastructure roles, with exposure to a range of technologies and practices.
