Backend Development

Introduction

Backend development focuses on the server side of web applications. It involves creating and managing databases, server logic, authentication, and APIs to ensure smooth communication between the server and client.

Key Technologies

  • Java
  • Spring Boot
  • Hibernate
  • MySQL
  • REST APIs

Projects :

  • Drug Distribution Management System: A full-stack application developed using Spring Boot (backend), MySQL (database), and React.js (frontend) to manage drug distribution efficiently. It includes RESTful API integration for smooth backend-frontend communication.
  • Temperature Conversion Application: Java-based application built with Spring Boot to convert temperatures between Celsius, Fahrenheit, and Kelvin, featuring precise backend processing for accurate results.
  • Guessing Game: A Java-based game where the backend generates random numbers for users to guess. The game provides feedback on each guess (too high/low) and tracks the number of attempts.
  • Contact Management System: Java and Spring Boot-based system to manage contacts. Users can add, update, and delete contacts with persistent storage for effective information management.
  • Sudoku Solver: Java application that uses a backtracking algorithm to solve Sudoku puzzles, with structured backend logic for efficient problem-solving.

For more details and to access these projects, please visit my GitHub repository.
